Step-by-Step Guide

  1. Enter the number of users at the beginning of the month.
  2. Enter the number of users at the end of the month.
  3. Enter the number of new user acquisitions during the month.
  4. Click the Calculate button to determine the Monthly Active Users (MAU).

Formula:

MAU = Users at Start + New User Acquisitions - Users at End
